   ```bash
        cd backend
        npm install
   ```

3. **Install frontend dependencies**

   ```bash
        cd ../frontend
        npm install
   ```

4. **Start the app**

- Start the backend:

  ```bash
      cd backend
      npm run dev
  ```

- Start the frontend:

  ```bash
      cd ../frontend
      npm start
  ```
